# Command Line

## Finding the right command
```
<Up>                Show the previous command
                    Pressing <Up> again will cycle backwards through the list

[partial]<CTRL-R>   Show the first command in the Search History that matches partial
                    Pressing <CTRL-R> again will cycle backwards through the matches
                    bck-i-search
                    Reverse Incremental Search
                    eg. cd<CTRL-R> - will show most recent command starting with "cd"

[partial]<CTRL-D>   Show all commands in the Search History that match that partial string
```

## Navigation
```
<CTRL-A>            Start of line

<CTRL-E>            End of line

<OPT-F>             Forward one word
<CTRL-F>            Forward one character

<OPT-B>             Backward one word
<CTRL-B>            Backward one character
```


## Edit
```
<CTRL-K>            Clear Line
```
